# Consistent Code Comment Decor

Want to keep your code comment decorations consistent? Look no further!

## Add This to Your Application

```JavaScript
// Here is a comment about some comments!
    fxCodeCommentLine =     "  //==========================================================================";
    fxCodeCommentBlock =    "  /*=========================================================================="
                            & Char(10) & Char(10) &
                            "   ==========================================================================*/";
```
